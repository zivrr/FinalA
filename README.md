# FinalA - OOP Assignment (Python)

This repository contains my FinalA OOP assignment solutions in Python.
The project is divided into 3 parts (each part is in a separate folder).

## Project Structure

- `encapsulation/`
  - Demonstrates OOP encapsulation principles.
  - Includes a main program to run the solution.

- `securities_inheritance/`
  - Demonstrates OOP inheritance with a base class and derived classes.
  - Includes a main program to run the solution.

- `repl_multiple_checker/`
  - A separate solution/program for the third part.
  - Includes a main program to run the solution.
